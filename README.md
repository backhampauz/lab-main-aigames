# lab-main-aigames

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能包含简单的演示、工具或交互式内容，不针对任何特定网站或域名。

## 项目简介

这是一个轻量级的 HTML 页面集合仓库。每个页面均为独立文件，可直接在浏览器中打开运行。仓库本身不提供服务器端功能，仅作为静态资源存储与分发的入口。

## 目录说明

- 仓库根目录下存放各个 HTML 页面文件。
- 页面命名遵循内容主题，便于识别。
- 部分页面可能依赖外部资源（如 CSS、JavaScript 库），这些资源通常通过 CDN 引用或内联在文件中。

## 页面归档说明

本仓库中的页面按以下原则归档：

- 每个页面独立存在，互不依赖。
- 归档时尽量保持文件结构简洁，避免嵌套目录。
- 页面内容以实用或演示为目的，不包含任何推广、引流或商业信息。
- 所有页面均可在本地直接运行，无需配置服务器环境。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有内容。
- 欢迎通过 Issues 提交问题或建议，但不保证及时回复。
- 不接受外部直接提交的 Pull Request，除非事先沟通。
- 仓库维护者保留对页面内容的最终解释权。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开任意 `.html` 文件。
3. 部分页面可能需要现代浏览器支持（如 ES6+、WebGL 等）。
